# Sea Street Detailing - Automated Scheduling Platform

Automating the booking system for my detailing business I started. This platform handles online scheduling, customer management, and payment processing for Sea Street Detailing.

## Tech Stack

### Frontend
- **Next.js 15** - React framework with App Router
- **React 19** - UI library
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Component library

### Backend
- **Next.js API Routes** - Serverless API endpoints
- **Prisma ORM** - Database toolkit and query builder
- **SQLite** - Embedded database (development)

### Integrations
- **Stripe** - Payment processing
- **Twilio** - SMS notifications and reminders
- **Google Calendar API** - Calendar synchronization

### Features
- Real-time availability checking
- Multi-step booking flow with validation
- Customer information management
- Service and add-on selection
- Automated SMS reminders
- Payment integration (on-site collection)
- Mobile-responsive design
- Admin dashboard
